# arp-scan-X

arp-scan on X-platform.

| | |
| --- | --- |
| o | Linux |
| o | BSD |
| o | MacOS |
| x | windows |



## Usage

### downloaad command
```command
$ go get github.com/HayatoDoi/arp-scan-x
```
